# Port-scanner
## To use this Port Scanner

Linux:
```bash
sudo git clone https://github.com/aflahsidhique/Port-scanner.git
```
Windows:
```bash
git clone https://github.com/aflahsidhique/Port-scanner.git
```
```
cd repo
```
## Usage
```
./port-scanner.sh [IP]
```
Example:
```bash
./port_scanner.sh 192.168.1.1
```
